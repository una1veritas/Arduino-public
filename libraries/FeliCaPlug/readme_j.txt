Arduino 用 FeliCaPlug ライブラリ ReadMe ファイル
===============================================
ソニー株式会社
2010年 9月1日

■ このライブラリについて
   
  本ライブラリは、Arduino から FeliCa Plug モジュールを操作するための
  ライブラリです。

■ 構成

   - FeliCaPlug.cpp, FeliCaPlug.h
     Arduino 用 FeliCaPlug ライブラリのソースコードです。

   - examples/
     ライブラリを使用したサンプルコードです。

   - FeliCaPlug-sjis.txt
     ライブラリのAPI仕様を記述したドキュメントです。

   - readme_j.txt
     このファイルです。

■ 使用許諾契約について

  このライブラリは「Arduino向けFeliCa Plug制御ライブラリ 使用許諾契約」
  で定められる条件の範囲で使用が許可されています。

  - 「Arduino向けFeliCa Plug制御ライブラリ 使用許諾契約」
     http://blog.felicalauncher.com/sdk_for_air/?page_id=2419
